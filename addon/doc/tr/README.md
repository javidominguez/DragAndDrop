# Sürükle ve Bırak

NVDA için nesneleri sürükleyip bırakmaya yönelik eklenti

## Kullanım

1. Sürüklemek istediğiniz nesneyi işaretleyin.
2. Fare imlecini veya nesne dolaşımını, bırakmak istediğiniz konuma taşıyın.
3. Bırakma seçeneğini seçin.

| Komut          | Masaüstü klavye | Dizüstü klavye | Açıklama                                                                                                                                                  |
| ------------------ | --------------- | -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| fareİmleçBilgisi    | NVDA+-          | NVDA+Control+, | Fare imlecinin konumu ve işaret ettiği nesne hakkında bilgi verir. İki kez basıldığında, sürükleyip bırakmak için işaretlenen nesne hakkında bilgi verir. |
| Sürüklenecek Nesneyi seçin | NVDA+,          | NVDA+,         | Gezgin içindeki nesneyi sürüklemek üzere işaretler.                                                                                                       |
| Sürükle ve Bırak        | NVDA+.          | NVDA+Shift+,   | Seçilen nesneyi sürükler. Ardından, nereye bırakılacağını seçmek için boşluk tuşuna, yön tuşlarına veya Enter tuşuna basın.                               |

Bunlar bırakma seçenekleridir:

* Fare imlecinin mevcut konumuna bırakmak için Enter tuşuna basın.
* Gezgin içindeki mevcut nesne üzerine bırakmak için Boşluk tuşuna basın.
* Gezgin içindeki mevcut nesnenin etrafına bırakmak için yön tuşlarına basın.
